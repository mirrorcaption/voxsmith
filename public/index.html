直连 Cloudflare API

填入你自己的 Account ID 和 API Token， 前端会绕过 Worker、直接调 api.cloudflare.com，使用你自己的额度跑 Whisper。 凭据只存在本浏览器，不会上传到服务器。

⚠️ Cloudflare Token 只勾 Workers AI Read 权限， 不要给 Edit / Account 级别。

想在直连模式下也用 refine（清理 / 润色 / 结构化）？ 额外填一个 Cerebras API Key，前端会直接调 api.cerebras.ai 跑 gpt-oss-120b 做后处理，同样不经过本服务器。留空就只输出原始转写。
